# Azure TranslateBot

AI-Powered translation web app using:

- Azure Translator API
- Azure Computer Vision (OCR)
- React.js Frontend
- Node.js/Express Backend
- Hosted on Azure App Service

# Team

- Rahil - Frontend + Azure Deployment
- Amrit - Backend + Translator API
- Peter - Backend + OCR API

# Project Structure

/backend - Express server with API routes
/frontend - React.js UI

## Installation

1. Install backend dependencies
   cd backend
   npm install

2. Install frontend dependencies
   cd frontend
   npm install

## .env with key and endpoint is in backend folder
